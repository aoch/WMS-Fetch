WMS-Fetch by aoch

Fetch Data from WMS via CLI, ANT, Maven to help create release notes etc.

WMS Fetch helps automate the retrieval of data from the Convergys Work Management System (WMS) in order to create things such as Release Notes (capturing list of fixed defects for a release).
